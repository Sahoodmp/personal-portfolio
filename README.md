# Personal Portfolio

A simple personal portfolio website built with React and Vite.

## Links
- **GitHub Repository:** [https://github.com/Sahoodmp/personal-portfolio.git]
- **Live Website:**    [https://transcendent-melomakarona-3b0612.netlify.app/]

## Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
